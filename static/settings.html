Pengaturan

WhatsZan Desktop Client

Umum
Jalankan Otomatis saat Startup
Nyalakan aplikasi secara otomatis ketika komputer dihidupkan
Sembunyikan dengan ESC
Tekan tombol ESC pada keyboard untuk menyembunyikan jendela
Tutup ke Tray
Menekan tombol tutup (X) hanya akan menyembunyikan jendela
Pilah Unduhan Otomatis
Simpan file ke folder Windows yang sesuai (Video, Gambar, dll)
Awalan Notifikasi
Teks awalan pada notifikasi sistem (misal: WhatsZan - Pesan Baru)
Privasi
Aktifkan Mode Rahasia
Mengaktifkan sensor pada layar. Tekan (Ctrl+Shift+B) kapan saja.
Blur Daftar Kontak
Blur Foto Profil (PP)
Blur Isi Pesan
Blur Media (Gambar & Video)
Tingkat Blur
Geser untuk mengatur seberapa buram efek sensornya
Contoh Pesan Rahasia
Hanya Kontak Spesifik
Jika diisi, opsi Daftar Kontak di atas hanya berlaku untuk nama-nama ini. Pisahkan dengan koma.
Incognito / Power User
Ghost Mode: Centang Biru
Mematikan laporan telah dibaca (Read Receipts). Lawan bicara tidak tahu pesannya sudah kamu baca.
Ghost Mode: Sembunyikan Mengetik
Menghilangkan indikator "Sedang mengetik..." di HP lawan bicaramu.
Anti-Hapus Pesan
Memulihkan pesan yang ditarik/dihapus oleh pengirim
Pengunduh Status (Status Saver)
Tampilkan tombol 'Simpan Status' saat melihat Story teman
Kamus Otomatis
Ganti kata pintasan menjadi teks panjang secara otomatis saat mengetik di obrolan.
Tampilan
Tampilkan Menu Bar
Menampilkan menu aplikasi di bagian atas jendela
Sembunyikan Menu Bar Otomatis
Menu bar hanya akan muncul saat tombol Alt ditekan
Pemeriksa Ejaan (Spellcheck)
Garis bawah merah akan muncul pada kata yang salah ketik
Lanjutan
Ekstensi Chrome
Pasang ekstensi browser seperti AdBlock atau Grammarly
User Agent Kustom
String identifikasi perangkat. Kosongkan untuk menggunakan default.
